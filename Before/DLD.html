Theorem 1 Idempotency ▼
(a)  x + x = x
(b)  x · x = x   ← dual (free!)
Idea: OR-ing or AND-ing a variable with itself gives back that variable — no new information is added. We'll prove (a) formally; (b) follows from duality.
Proof of (a): x + x = x
1
x + x
Start with the left-hand side.
2
= (x + x) · 1
Identity Law — multiplying by 1 changes nothing: A·1 = A
3
= (x + x) · (x + x')
Complementarity — x + x' = 1 (a variable OR-ed with its complement is always 1)
4
= x + (x · x')
Distributive Law — factor out x from both terms: x·(A+B) = x·A + x·B applied in reverse
5
= x + 0
Complementarity — x · x' = 0 (a variable AND-ed with its complement is always 0)
6
= x   ✓
Identity Law — x + 0 = x
💡 Dual (b) for free: Swap + ↔ · and 0 ↔ 1 everywhere in the proof above, and you get the proof of x·x = x without any extra work.
Theorem 2 Null Elements (Annihilators) ▼
(a)  x + 1 = 1
(b)  x · 0 = 0   ← dual (free!)
Idea: OR-ing anything with 1 always gives 1 — because 1 dominates. AND-ing anything with 0 always gives 0.
Proof of (a): x + 1 = 1
1
x + 1
Start with the left-hand side.
2
= x + (x + x')
Complementarity — replace 1 with (x + x'), since x + x' = 1
3
= (x + x) + x'
Associativity — re-group the OR operations
4
= x + x'
Idempotency (Theorem 1a) — x + x = x
5
= 1   ✓
Complementarity — x + x' = 1
💡 Dual (b) for free: Swap + ↔ · and 1 ↔ 0 to get x · 0 = 0.
Theorem 3 Involution (Double Complement) ▼
(x')' = x
Idea: Complementing twice returns the original value. This is self-dual — there's no separate dual form.
1
(x')' = ?
We want to find what (x')' equals.
2
x' + x = 1  and  x' · x = 0
Complementarity — the complement of x' must satisfy both these conditions.
3
x satisfies both: x' + x = 1 ✓ and x' · x = 0 ✓
x itself already satisfies both complementarity axioms with respect to x'.
4
Therefore (x')' = x   ✓
Uniqueness of complement — complements are unique, so x must be the complement of x'.
Theorem 4 Absorption ▼
(a)  x + x·y = x
(b)  x·(x + y) = x   ← dual (free!)
Idea: x "absorbs" the term x·y because x already covers all cases where x·y is 1. If x = 1, the whole expression is 1. If x = 0, x·y = 0 too. So y is irrelevant.
Proof of (a): x + x·y = x

x + x·y
Start with the left-hand side.
2
= x·1 + x·y
Identity Law — x = x·1 (multiplying by 1 doesn't change anything)
3
= x·(1 + y)
Distributive Law — factor out x
4
= x·1
Null Element (Theorem 2a) — 1 + y = 1
5
= x   ✓
Identity Law — x·1 = x
💡 Dual (b) for free: Swap + ↔ · and 1 ↔ 0 throughout to get x·(x+y) = x.
Theorem 5 Absorption (Variant) ▼
(a)  x + x'·y = x + y
(b)  x·(x' + y) = x·y   ← dual (free!)
Idea: When x is 0 (so x' = 1), the term x'·y becomes just y — and x+y = y. When x is 1, x+x'·y = 1 = x+y. So in all cases, x + x'·y = x + y.
Proof of (a): x + x'·y = x + y
1
x + x'·y
Start with the left-hand side.
2
= (x + x')·(x + y)
Distributive Law — OR distributes over AND: A + B·C = (A+B)·(A+C)
3
= 1·(x + y)
Complementarity — x + x' = 1
4
= x + y   ✓
Identity Law — 1·A = A
💡 Dual (b) for free: Swap + ↔ · and 1 ↔ 0 to get x·(x' + y) = x·y.
Theorem 6 DeMorgan's Theorem ⭐ ▼
(a)  (x + y)' = x'·y'
(b)  (x·y)' = x' + y'   ← dual (free!)
Big idea: To prove that two expressions are complements of each other, we only need to verify two things:
① Their OR = 1    ② Their AND = 0

We'll show that x'·y' is the complement of (x+y) by checking both conditions.
Proof of (a): (x + y)' = x'·y'
Condition ①: (x + y) + x'·y' must equal 1
1
(x + y) + x'·y'
We need to show this equals 1.
2
= x + y + x'·y'
Associativity — remove unnecessary parentheses from (x+y)
3
= (x + x'·y') + (y + x'·y')
Wait — use a smarter route. Go back to step 2 and apply Absorption Variant.
4
x + y + x'·y' = (x + y + x')·(x + y + y')
Distributive Law — A + B·C = (A+B)·(A+C), with A=(x+y)
5
= (x + x' + y)·(x + y + y')
Commutativity — reorder terms within each factor
6
= (1 + y)·(x + 1)
Complementarity — x + x' = 1 and y + y' = 1
7
= 1·1 = 1   ✓
Null Element — 1 + anything = 1
Condition ②: (x + y) · x'·y' must equal 0
1
(x + y) · (x'·y')
We need to show this equals 0.
2
= x·x'·y' + y·x'·y'
Distributive Law — distribute (x+y) over x'·y'
3
= 0·y' + x'·0
Complementarity — x·x' = 0 and y·y' = 0
4
= 0 + 0 = 0   ✓
Null Element — 0 · anything = 0
Conclusion
Since (x + y) + x'·y' = 1 AND (x + y) · x'·y' = 0, by the uniqueness of complement, we conclude that (x + y)' = x'·y'. ✓

Theorem 7 Consensus Theorem ⭐ ▼
(a)  x·y + x'·z + y·z = x·y + x'·z
(b)  (x+y)·(x'+z)·(y+z) = (x+y)·(x'+z)   ← dual (free!)
Big idea: The term y·z is redundant — it's called the consensus term. Any time y·z = 1, either x·y = 1 (when x=1) or x'·z = 1 (when x=0). So y·z never adds new information.

We prove that x·y + x'·z + y·z = x·y + x'·z by showing the left side simplifies to the right.
Proof of (a): x·y + x'·z + y·z = x·y + x'·z
1
x·y + x'·z + y·z
Start with the left-hand side. We want to eliminate y·z.
2
= x·y + x'·z + y·z·1
Identity Law — multiply y·z by 1 (no change)
3
= x·y + x'·z + y·z·(x + x')
Complementarity — replace 1 with (x + x'), since x + x' = 1
4
= x·y + x'·z + x·y·z + x'·y·z
Distributive Law — distribute y·z over (x + x')
5
= (x·y + x·y·z) + (x'·z + x'·y·z)
Commutativity / Re-grouping — pair the x terms and x' terms together
6
= x·y·(1 + z) + x'·z·(1 + y)
Distributive Law — factor x·y from the first group and x'·z from the second
7
= x·y·1 + x'·z·1
Null Element — 1 + z = 1 and 1 + y = 1
8
= x·y + x'·z   ✓
Identity Law — A·1 = A. The consensus term y·z has been eliminated!

💡 Why is y·z called the "consensus term"?
Think of x as a "voter". When x=1: x·y covers the case. When x=0: x'·z covers the case. The consensus of both voters already covers all situations where y=1 AND z=1 — so y·z is always redundant and can be dropped.

💡 Dual (b) for free: Swap + ↔ · and 1 ↔ 0 throughout to get the POS form of the consensus theorem.
Principle Duality Principle ▼
If a Boolean equation is valid, its dual is also valid.
Dual = swap  +↔·  and  0↔1  everywhere.
Idea: Boolean algebra is symmetric between AND/OR and 0/1. Every axiom has a dual, so any theorem derived from axioms also has a dual theorem — automatically valid, no separate proof needed.
Example: From DeMorgan (a) to DeMorgan (b)
1
Start:  (x + y)' = x'·y'
This is DeMorgan's theorem (a) — proven above.
2
Swap + ↔ · everywhere:
Replace every OR (+) with AND (·) and vice versa.
3
Result:  (x · y)' = x' + y'   ✓
This is DeMorgan (b) — obtained for free via duality!
Another Example
1
Start:  x + 1 = 1
Null element theorem (a).
2
Swap + → · and 1 → 0:
Apply the duality transformation.
3
Result:  x · 0 = 0   ✓
Null element theorem (b) — free from duality.
⚠️ Important: Duality swaps operators and constants, but does NOT complement the variables. x stays as x (not x').
