Q2 — 4-bit Adder/Subtractor
5 Marks

Circuit Design & Role of Mode Bit M

A 4-bit Adder/Subtractor uses a 4-bit ripple carry adder with XOR gates on the B inputs, controlled by a mode select bit M.

Key Principle
Each B bit is XORed with M:
• M = 0 (Addition): B⊕0 = B → adder computes A + B, Cin = 0
• M = 1 (Subtraction): B⊕1 = B' → adder computes A + B' + 1 = A + (2's complement of B) = A − B
Fig: 4-bit Adder/Subtractor — M=0 → Addition, M=1 → Subtraction
✓ Carry Propagation: In a ripple carry adder, each full adder's Cout feeds into the next stage's Cin. When M=1 (subtraction), Cin=1 completes the 2's complement, and a final Cout=1 indicates no borrow (positive result).
Q3 — 3-to-8 Decoder Implementation
5 Marks

Implement simultaneously: F1(A,B,C) = Σm(1,3,5,7) and F2(A,B,C) = Σm(0,2,6,7) using a single 74x138 (3-to-8 decoder) and external OR gates.

74x138 Key Facts

74x138 Outputs
The 74x138 has active-LOW outputs: output Ȳᵢ = 0 when minterm i is selected.
Enable: G1=1 (active HIGH), G2A=0, G2B=0 (active LOW) — all must be satisfied.
To implement F using active-LOW outputs: use a NAND gate instead of OR, OR connect to a bubbled-input OR (= NAND).
Simplification: Since 74x138 outputs are active-LOW, connecting them to a NAND gate gives the correct active-HIGH output of the OR function.

F1 = Σm(1,3,5,7) — Connection

1
Minterms: 1, 3, 5, 7 → connect outputs Ȳ1, Ȳ3, Ȳ5, Ȳ7 to a 4-input NAND gate.
2
NAND of active-LOW lines = OR of corresponding minterms → F1 = m1 + m3 + m5 + m7
3
Insight: F1 = C (these are all odd minterms — C=1). You could also just wire C directly.

F2 = Σm(0,2,6,7) — Connection

1
Minterms: 0, 2, 6, 7 → connect outputs Ȳ0, Ȳ2, Ȳ6, Ȳ7 to a 4-input NAND gate.
2
F2 = m0 + m2 + m6 + m7
FunctionMintermsDecoder Outputs (NAND inputs)Expression
F11, 3, 5, 7Ȳ1, Ȳ3, Ȳ5, Ȳ7= C
F20, 2, 6, 7Ȳ0, Ȳ2, Ȳ6, Ȳ7= A'B'C' + A'BC' + ABC' + ABC = B'C' + AB
✓ Key Point: The 74x138 active-LOW outputs feed into NAND gates. A NAND of active-LOW signals = OR of the corresponding minterms (by De Morgan's theorem). Ȳ7 is shared between both F1 and F2 — connect it to both NAND gates.

↑ = rising edge. Q toggles every clock cycle (T flip-flop behavior). Period of Q = 2× clock period.
✓ Toggle Behavior: With J=K=1, the JK flip-flop toggles on every active clock edge. This is equivalent to a T flip-flop with T=1. The output Q has exactly half the frequency of the clock signal.
Q6 — Synchronous Mod-6 Counter
5 Marks

Design a synchronous Mod-6 counter (counts 0→1→2→3→4→5→0) using JK flip-flops.

We need 3 flip-flops (Q2 Q1 Q0) since 2³=8 ≥ 6. States 6 and 7 are unused — reset on state 6 (110).

K-map Simplification

Using the JK excitation table (J=0,K=X for no change; J=1,K=X to set; J=X,K=1 to reset):

Simplified Expressions (from K-maps)
J2 = Q1·Q0    K2 = Q0
J1 = Q2'·Q0    K1 = Q0
J0 = Q2' + Q1' = (Q2·Q1)'    [= 1 when NOT in state 6]   K0 = 1
✓ Reset Condition: When Q2=1 and Q1=1 (state 6 or 7), J0 is driven low and the CLR/reset line forces the counter back to 000. In practice, detect Q2·Q1=1 with an AND gate and feed it to the synchronous CLR inputs of all flip-flops.

State Diagram

Logic Circuit Summary

1
Three negative-edge JK flip-flops (FF2, FF1, FF0) clocked simultaneously.
2
AND gate: detects Q2·Q1 = state 6. Output feeds asynchronous CLR of all three FFs → instant reset to 000.
3
Connect J/K inputs per equations above. K0=1 (tie HIGH). All FFs share the same clock.
✓ Synchronous Reset Alternative: For a fully synchronous design, decode state 6 and feed it back as a synchronous clear input, so the reset happens on the next clock edge (counter goes 0→1→2→3→4→5→0, with state 6 never appearing in steady state).
